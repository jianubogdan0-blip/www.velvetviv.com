# www.velvetviv.com

Personal streaming landing page.

## Setup

1. **Add your background photo** — place your image in this folder and name it `photo.jpg`
   (or change the `backgroundPhoto` variable in the `EASY-EDIT SECTION` of `index.html`).

2. **Update the goal progress** — open `index.html` and find the `EASY-EDIT SECTION` block.
   Change `currentRaised` to the current amount received.

3. **Update top tippers** — in the same block, add entries to the `topTippers` array:
   ```js
   { name: "username", amount: 500 },
   ```

4. **Change schedule hours** — edit the `<span class="time">` values in the schedule section.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page |
| `style.css` | Styles |
| `photo.jpg` | Your background photo (you must add this) |
